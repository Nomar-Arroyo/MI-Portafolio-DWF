# 🎓 Plataforma de Perfil Estudiantil — React + Vite

> Aplicación web modular desarrollada con **React**, **Vite** y **React Router DOM** para gestionar perfiles estudiantiles, autenticación simulada y visualización dinámica de proyectos académicos.

---

# 📘 Descripción General

¡Bienvenido al repositorio del proyecto **Perfil de Estudiante**!

Esta aplicación web moderna implementa una arquitectura basada en componentes reutilizables utilizando React y Vite como entorno de desarrollo ultrarrápido.

La plataforma incluye:

- Sistema de rutas dinámicas
- Flujo de autenticación simulado
- Gestión visual de perfiles
- Carga de imágenes en tiempo real
- Vista detallada de proyectos académicos
- Diseño responsive moderno

# 🚀 Guía de Ejecución Local

## 1️⃣ Navegar al Proyecto

```bash
cd Pagina-de-Perfil-de-Estudiante
```

## 2️⃣ Instalar Dependencias

```bash
npm install
```

## 3️⃣ Iniciar el Servidor

```bash
npm run dev
```

## 4️⃣ Abrir en el Navegador

```text
http://localhost:5173/
```
