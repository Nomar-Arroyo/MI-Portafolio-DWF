# 📘 Descripción General

Bienvenido a mi portafolio de evidencias del segundo trimestre de **Desarrollo Web Frontend Esencial**.

Este repositorio recopila proyectos, evaluaciones y ejercicios prácticos desarrollados durante el proceso de aprendizaje, evidenciando una progresión técnica desde:

- HTML5 semántico
- CSS3 moderno
- JavaScript ES6+
- Arquitecturas modulares
- React y Vite
- Git y GitHub
- Desarrollo asistido por Inteligencia Artificial

El objetivo es demostrar la evolución continua de conocimientos técnicos mediante proyectos funcionales y documentados.

#### 🧠 Capa de Estado

- Administración de memoria.
- Fuente única de verdad.

#### 🎨 Capa de UI

- Renderizado controlado.
- Actualización eficiente de componentes.

### Tecnologías
