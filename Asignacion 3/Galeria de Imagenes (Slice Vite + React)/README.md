# 🌌 Windows Gallery Ultra Pro — React + Vite

> Aplicación web moderna desarrollada con **React 18** y **Vite**, enfocada en la gestión avanzada de galerías de imágenes mediante una arquitectura modular y escalable.

---

# 📘 Descripción General

El proyecto fue refactorizado desde una arquitectura monolítica basada en HTML y JavaScript Vanilla hacia una aplicación moderna construida con React y Vite.

## Características principales

- Gestión de usuarios y sesiones.
- Administración dinámica de álbumes.
- Visualización interactiva de imágenes.
- Diseño Glassmorphism moderno.
- Persistencia local mediante LocalStorage.
- Arquitectura desacoplada y modular.

# 🚀 Instalación

## Clonar el repositorio

```bash
git clone <URL_DEL_REPOSITORIO>
cd windows-gallery-ultra-pro
```

## Instalar dependencias

```bash
npm install
```

## Ejecutar el proyecto

```bash
npm run dev
```

Abrir:

```text
http://localhost:5173
```

## Reconstrucción limpia

```bash
npm run dev -- --force
```
